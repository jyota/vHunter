mapedit.py

A tile map editor

ESC: Save map file name
s key: Save files
F1: Save script file name
d: delete tile
,: decrease script item selection
.: increase script item selection
z: place script item
x: remove script item
F2: Select current tile for brush
q key: Quit

When Save file name is not initialized by the user, program will save current map data to "default.map"-- currently will not automatically use a loaded filename in order to be safe and avoid errors.
